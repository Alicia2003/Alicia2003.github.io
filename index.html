Random Quote Generator

For this project we were assigned to create a web page that shows a random motivational quote when a button is clicked. We used arrays, math.random() method, jQuery click functions, and the .length method to display different quotes from the array onto the webpage when the button was clicked.

Character Quiz

For this project we used our skills in jQuery and javascript to create a webpage were users can take a quiz to find out which charcater they are from a tv show. We used else if statements and compound conditionals to display a specific character after certain inputs were put in.

Photo Gallery

For this project we were assigned to create a web page were users could paste links into the input fields to display images onto the webpage. We used arrays, jQuery click functions, input .val(), .push, .length, and foreach loops to display images and the input field.
